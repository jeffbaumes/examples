ENCODE Genome Data

Note: Data may take 30 seconds or so to load. Works best in the latest versions of Chrome and Safari.

This visualization displays human gene annotation data from the ENCODE project. Each of the 167,536 annotations are displayed in linear order, grouped by chromosome. Blue indicates an annotation is encoded on the forward strand, while orange annotations are on the reverse strand. Color intensity shows the length of each annotation, with the strongest colors containing 10,000 or more base pairs. Hovering over the data shows the related gene for each annotation. Clicking on an annotation opens a new page with details on the gene.

Source: ENCODE Genes V12
